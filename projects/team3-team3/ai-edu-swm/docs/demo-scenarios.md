# Demo Scenarios

## 대학생 시나리오

- 날짜: 2026-06-03
- 하루 시간: 09:00~23:00
- 고정 일정:
  - 10:00~12:00 전공 수업
  - 14:00~15:00 팀플 회의
  - 18:00~19:00 저녁 약속
- 작업:
  - 알고리즘 과제, 120분, 중요도 5, 오늘 마감, 분할 가능, deep
  - 영어 단어 암기, 30분, 중요도 2, 분할 가능, light
  - 팀플 자료 정리, 60분, 중요도 4, 내일 마감, light

기대 결과:

- 오늘 마감인 알고리즘 과제를 긴 block에 먼저 배치한다.
- 짧은 block에는 영어 단어 암기 같은 light work를 배치한다.
- 시간이 부족하면 낮은 우선순위 작업을 미배치로 표시한다.
